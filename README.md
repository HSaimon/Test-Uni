# Test-Uni

## Descrição
Este repositório contém um projeto Java focado em testes unitários, especificamente para a avaliação AV1. Ele demonstra a aplicação de testes unitários em um contexto de desenvolvimento Java.

## Tecnologias Utilizadas
- **Java**: Linguagem de programação principal.
- **Maven**: Ferramenta de automação de build e gerenciamento de dependências, utilizada para gerenciar o projeto `TestUnitarioAV1`.

## Estrutura do Projeto
A estrutura do projeto é organizada da seguinte forma:
- `TestUnitarioAV1/`: Diretório principal do projeto Java.
  - `pom.xml`: Arquivo de configuração do Maven para o projeto `TestUnitarioAV1`.
  - `src/`: Contém o código-fonte e os testes do projeto.
- `README.md`: Este arquivo.

## Instalação e Execução
Para configurar e executar o projeto localmente, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/HSaimon/Test-Uni.git
   cd Test-Uni
   ```

2. **Navegue até o diretório do projeto Maven:**
   ```bash
   cd TestUnitarioAV1
   ```

3. **Compile o projeto com Maven:**
   ```bash
   mvn clean install
   ```

4. **Execute os testes unitários:**
   ```bash
   mvn test
   ```
